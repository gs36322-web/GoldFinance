# 💰 Sistema de Controle Financeiro

Um sistema web simples e moderno para controle de finanças pessoais.  
O objetivo do projeto é ajudar usuários a registrarem **entradas e gastos**, acompanharem o **saldo total** e visualizarem os dados através de **gráficos e histórico de transações**.

Este projeto faz parte da minha jornada de aprendizado em **programação e desenvolvimento web**.

---

# 🚀 Funcionalidades

✅ Sistema de login  
✅ Dashboard financeiro  
✅ Registro de entradas e saídas  
✅ Histórico de transações  
✅ Separação visual entre receitas e gastos  
✅ Gráficos financeiros interativos  
✅ Filtro de transações por mês  
✅ Interface simples e intuitiva

---

# 📊 Como funciona

1️⃣ O usuário faz login no sistema  
2️⃣ Acessa o dashboard financeiro  
3️⃣ Registra entradas ou gastos  
4️⃣ O sistema calcula automaticamente o saldo  
5️⃣ As transações aparecem no histórico  
6️⃣ Os gráficos mostram a análise financeira
